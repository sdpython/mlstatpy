
.. index:: glossary

Glossary
========

.. glossary::

    ml
        Raccourci pour `Machine Learning <https://en.wikipedia.org/wiki/Machine_learning>`_.

    NLP
        Raccourci pour `Natural Language Processing <https://en.wikipedia.org/wiki/Natural_language_processing>`_.

    outlier
        Point aberrant. C'est une observation qui est à l'écart des autres observations
        du jeu de données de façon évidente. Il perturbe l'apprentissage d'une prédiction.
